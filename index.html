How to Use It?

Intro

We’ll show you how you can use SimpliSearch!

The Basics

It all starts with the search() function. It takes an object with two main properties: query and list. The query will be compared to each item in the list.

Conditions

Use these properties to add conditions:

  • minLength: Filters only if the query length is greater or equal to this value
  • maxResults: Limits the number of results
  • maxScore: Includes items only if their score is below this value

Formatting

Need to standardize your text format? Use the formatTo property to convert, for example, "hello world" into "hello-world" or "helloWorld".

Targets

If your list contains objects, use the targets property to specify which object keys to compare against the query.

Fallback

If no results match, search() returns an empty array. You can provide a fallback property with content to use instead.

onFinish

Use the onFinish method to receive results, the formatted query, and concated results.

Concatenation

Use the concat property like .map() to build HTML strings, enhance results, or transform output.
